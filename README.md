# MyLittlePhotobook

Demo App using Next.ts (Focused on Techniques below)

1. Next.js
2. SSR
3. SSG
4. CSS Tricks (position, display, grid, etc..)
5. Typescript

## Installation

Use `yarn` or `npm install` to install modules

## Usage

```bash
# Development mode
yarn dev
# Publish mode
yarn build && yarn start
```
